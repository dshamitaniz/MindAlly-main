# Audio Files for Nature Sounds

This directory should contain the following MP3 files for the Indian Nature Sounds meditation:

- `forest.mp3` - Forest sounds with birds and rustling leaves
- `river.mp3` - Gentle river flowing sounds
- `monsoon.mp3` - Monsoon rain sounds
- `birds.mp3` - Morning bird songs
- `ocean.mp3` - Ocean wave sounds

## How to add audio files:

1. Download or record nature sound MP3 files
2. Name them according to the list above
3. Place them in this `/public/audio/` directory
4. The meditation app will automatically use them

## Recommended sources for free nature sounds:

- Freesound.org
- Zapsplat.com
- YouTube Audio Library
- BBC Sound Effects Library

## File requirements:

- Format: MP3
- Duration: At least 2-3 minutes (will loop automatically)
- Quality: 128kbps or higher
- No copyright restrictions
